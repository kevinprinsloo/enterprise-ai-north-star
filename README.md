# Enterprise AI North Star

Source files for a GitHub Pages-hosted long-form article.

## Local preview

GitHub Pages uses Jekyll. With Ruby and Bundler installed:

```shell
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.

## Publishing

1. Create the `enterprise-ai-north-star` GitHub repository and push this folder to its default branch.
2. In the repository, open **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the default branch and `/ (root)`, then save.
5. After deployment, use the public page URL with Medium's **Import a story** feature.

Production URL: `https://kevinprinsloo.github.io/enterprise-ai-north-star/`

Review the imported Medium draft before publishing, especially its title, subtitle, headings, image, caption, alt text, table, code blocks and section breaks.
